<Heading level="2">
    Common Examples
</Heading>

<Paragraph>
    Here are some common use cases to get you started:
</Paragraph>

<BulletedList>
    <Mark bold>Array operations:</Mark> <Mark>_.map([1, 2, 3], n => n * 2)</Mark> returns <Mark>[2, 4, 6]</Mark>
</BulletedList>
<BulletedList>
    <Mark bold>Object operations:</Mark> <Mark>_.get(obj, 'nested.property', 'default')</Mark>
</BulletedList>
<BulletedList>
    <Mark bold>Collection filtering:</Mark> <Mark>_.filter(users, { 'active': true })</Mark>
</BulletedList>
<BulletedList>
    <Mark bold>Function composition:</Mark> <Mark>_.flow(_.add(1), _.multiply(2))(5)</Mark> returns <Mark>12</Mark>
</BulletedList>

<Callout icon="💡" blockColor="yellow" borderColor="light_orange">
    Lodash methods are chainable. Use <Mark>_.chain()</Mark> to enable explicit chaining or use implicit chaining with methods that return wrapped values.
</Callout>
