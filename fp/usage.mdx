<Heading level="2">
    Tips for Migration
</Heading>

<Paragraph>
    If you're migrating from standard Lodash to FP module:
</Paragraph>

<BulletedList>
    Remember the argument order: <Mark>iteratee-first, data-last</Mark>
</BulletedList>
<BulletedList>
    FP methods don't mutate data - use the returned value
</BulletedList>
<BulletedList>
    Take advantage of auto-currying for partial application
</BulletedList>
<BulletedList>
    Use <Mark>fp.__</Mark> placeholders for flexible argument ordering
</BulletedList>
<BulletedList>
    Consider using <Mark>fp.flow</Mark> for composing multiple operations
</BulletedList>
