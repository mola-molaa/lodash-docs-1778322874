<Heading level="1">
    Introduction to Lodash
</Heading>

<Paragraph>
    <Mark bold>Lodash</Mark> is a modern JavaScript utility library delivering modularity, performance & extras. It makes JavaScript easier by taking the hassle out of working with arrays, numbers, objects, strings, etc.
</Paragraph>

<Paragraph>
    Lodash's modular methods are great for:
</Paragraph>

<BulletedList>
    Iterating arrays, objects, & strings
</BulletedList>
<BulletedList>
    Manipulating & testing values
</BulletedList>
<BulletedList>
    Creating composite functions
</BulletedList>

<Heading level="2">
    Why Choose Lodash?
</Heading>

<Paragraph>
    Lodash provides a consistent API across different JavaScript environments and handles edge cases that native methods might not cover. It's widely used in production applications and is maintained by a dedicated team.
</Paragraph>

<Heading level="2">
    Module Formats
</Heading>

<Paragraph>
    Lodash is available in a variety of builds & module formats:
</Paragraph>

<BulletedList>
    <Mark bold>lodash</Mark> & per method packages - Main library
</BulletedList>
<BulletedList>
    <Mark bold>lodash-es</Mark> - ES modules version
</BulletedList>
<BulletedList>
    <Mark bold>lodash/fp</Mark> - Functional programming build
</BulletedList>
<BulletedList>
    <Mark bold>lodash-amd</Mark> - AMD format
</BulletedList>

<Callout icon="ℹ️" blockColor="blue" borderColor="blue">
    Lodash is released under the MIT license & supports modern environments. Review the build differences & pick one that's right for you.
</Callout>
